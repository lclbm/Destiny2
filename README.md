## 💛初次购买：邀请小日向进群
1. 付款后将**交易截图**发送给群主，另外再将你的群号以**纯数字**的方式发送给群主
2. 再在**你的群**中按照以下步骤操作：
   1. 选择邀请成员
   2. 从群聊中选择
   3. 选择小日向交流群
   4. 选择**小日向5号机**（管理员，粉色的二次元头像）
3. 邀请完成后，**小日向5号机**会在**小日向交流群**内@你，如果未收到请稍等5-10分钟
   * 如果10分钟后小日向任然没有@你，请尝试重新邀请小日向
4. 小日向顺利进群后发送<kbd>**d2**</kbd>即可查看菜单
